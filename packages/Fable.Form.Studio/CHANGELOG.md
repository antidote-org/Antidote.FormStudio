## 3.0.0 - 2022-03-28

### Changed

* Fix #24: Allows to customise the actions of the form.

    Here is how to migrate your old code:

    ```diff
        {
            Dispatch = dispatch
            OnChange = FormChanged
    -       Action = "Sign in"
    +       Action = Form.View.Action.SubmitOnly "Sign in"
            Validation = Form.View.ValidateOnSubmit
        }
    ```
